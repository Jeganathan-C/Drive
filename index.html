To enable Google login and real Gmail OTP, you need to provide your API credentials. These are stored only in your browser — never sent anywhere.

⚠ Demo mode available: Skip setup to use email/password login with a simulated OTP (shown on screen). For real Google OAuth + Gmail OTP, fill in the credentials below.
1 Google OAuth Client ID
Get from Google Cloud Console → Create OAuth 2.0 Client → Web Application. Add your page URL to "Authorized JavaScript origins".
2 EmailJS Settings (for Gmail OTP)
Free at emailjs.com. Connect your Gmail account as a service. Create a template with variables: {{otp}}, {{to_email}}, {{user_name}}.
